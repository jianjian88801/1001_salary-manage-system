# 基于ssm实现的工资管理系统

#### 介绍
基于ssm实现的工资管理系统，前端采用jsp+layui，后端为ssm，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及企业级项目的入门。


#### 软件架构
前端：jsp | jquery | layui  
后端：spring | springmvc | mybatis  
环境：jdk1.8+ | mysql | maven
